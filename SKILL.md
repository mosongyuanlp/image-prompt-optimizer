# AI 图片生成提示词优化器

## 核心原理

AI 图片生成的安全审查**不是简单的关键词过滤**，而是一个**画面风险预测系统**。它综合判断五个层面：

1. **提示词文本风险** — 单个词语在上下文中的风险
2. **整体语义意图** — 提示词整体指向什么类型的画面
3. **风险元素叠加** — 多个中等风险元素组合后的总风险
4. **生成过程方向控制** — 模型在生成中会被安全系统拉向保守
5. **最终图像审查** — 即使文字通过，最终图片也可能被拦截

所以同一个词有时能过、有时不过，取决于**整段提示词最终指向什么画面**。

## 改写公式

将用户的原始意图通过以下管线转化为安全稳定的提示词：

```
原始欲望/描述
  ↓ 替换为气质词
  ↓ 整体体态描述（非局部强调）
  ↓ 服装得体化（写质感，不写暴露度）
  ↓ 场景明亮化（高级、干净、商业化）
  ↓ 镜头商业化（时尚杂志、Lookbook、editorial）
  ↓ 加安全边界声明
  = 审查友好的提示词
```

## 意图判断公式

用这个公式判断一条提示词是否稳定：

```
人物是谁 + 穿什么 + 在哪里 + 做什么动作 + 镜头怎么看 + 画面重点是什么
= 系统理解的生成意图
```

**稳定方向示例：** 成年女性 + 得体连衣裙 + 明亮室内 + 自然坐姿 + 平视镜头 + 重点是气质和服装 → 安全概率高

**高风险方向示例：** 年轻女性 + 暴露泳装 + 昏暗卧室 + 挑逗姿势 + 低机位 + 重点是胸臀 → 风险概率高

## 工作流程

### 步骤一：分析用户输入

用户可能提供以下任一形式的输入：
- 一段已有的提示词（需要优化/改写）
- 一个模糊的生成意图描述（需要转写成完整提示词）
- 一个生成失败的提示词（需要诊断问题并改写）

首先判断输入类型，然后进入对应流程。

### 步骤二：风险扫描

对提示词进行逐项检查，参考 `references/word-lists.md` 中的红黄绿词库：
- 标出所有红色词（必须删除或替换）
- 标出所有黄色词（需要转译为更高级的表达）
- 检查是否存在风险元素叠加（多个中等风险词组合）

同时检查以下五大高频触发原因：

| 触发原因 | 检查要点 |
|---|---|
| 年龄不明确 | 是否出现「少女/萝莉/幼态」，是否与「性感/挑逗」组合 |
| 身体局部强调 | 是否单独强调胸/臀/局部特写 |
| 挑逗姿势 | 是否有低机位/前倾/撩人/挑逗动作 |
| 私密场景 | 是否有昏暗卧室/浴室/暧昧氛围 |
| 过度暴露服装 | 是否写暴露度而非服装质感 |

### 步骤三：执行改写

按照改写公式逐层转化。核心替换规则（完整词库见 `references/word-lists.md`）：

**高频替换对照：**
- 性感 → 高级女性美 / 成熟吸引力
- 魅惑 → 神秘优雅 / 眼神有故事感
- 身材火辣 → 身形圆润匀称 / 曲线自然流畅
- 丰满 → 健康丰腴 / 成熟体态
- 撩人 → 有吸引力但克制
- 贴身 → 剪裁合身 / 版型贴合
- 暧昧 → 柔和梦幻 / 情绪细腻

**结构模板（按需选用）：**

```
生成一张[年龄]岁成年[地域]女性角色[风格] [画幅]构图。

画面中是一位[气质描述]的女性角色，[位置构图]。[姿态描述]。

人物[五官描述]，[肤质描述]。[整体体态描述——从肩到腰到整体曲线，写整体不写局部]。

服装为[颜色][款式]，[剪裁描述]，[材质描述]。画面重点表现[气质/姿态/服装质感/光影]。

背景为[明亮/干净的场景]，[光线描述]，[氛围描述]。
```

### 步骤四：八点自检

改写完成后，逐项核对以下清单，全部通过才输出：

1. ✅ 是否明确写了"成年女性角色"？
2. ✅ 是否避免了"少女感 + 性感"的组合？
3. ✅ 是否没有单独强调胸部、臀部等身体局部？
4. ✅ 是否没有使用挑逗、诱惑、勾引等词？
5. ✅ 是否没有低机位、身体凝视、局部特写？
6. ✅ 服装是否写成得体、合身、剪裁优雅？
7. ✅ 场景是否明亮、干净、商业化，而不是私密暧昧？
8. ✅ 画面重点是否放在气质、姿态、服装、光影、角色设计上？

如有任何一项未通过，返回步骤三修正。

### 步骤五：输出

输出内容包括：

1. **优化后的完整提示词**（可直接复制使用）
2. **改动说明**（简要列出主要改动及原因，帮助用户理解逻辑）
3. **可选：负面提示词**（如果用户频繁被拦截，附加一段安全边界声明）

**负面提示词模板（按需附加）：**

标准版：
```
避免裸露，避免挑逗动作，避免低俗化表达，避免身体部位特写，
避免低机位凝视视角，避免未成年感，避免过度暴露服装。
整体以高级角色设计、时尚人像、商业视觉和审美表达为主。
```

简洁版：
```
画面保持高级、得体、克制，不做挑逗姿态，不突出身体局部，
不使用低俗或成人向表达。
```

强约束版：
```
重点表现人物气质、服装质感、姿态美感、光影层次和角色设计，
不以身体暴露作为画面重点。
```

## 常见场景快速处理

### 场景：泳装

将「性感泳装美女」类描述转化为「高端泳装品牌 Lookbook」框架：强调产品展示、度假氛围、商业目录感。

### 场景：女仆/甜美风

将「性感女仆」转化为「幻想系角色美术设计」框架：甜美梦幻主题服饰、角色美术感、气质成熟温柔。

### 场景：回眸姿势

将「性感回眸」转化为「时尚人像」框架：身体轻微侧转、头部自然回望、肩颈线条舒展、高级感。

### 场景：S 型身材

将「胸大腰细臀翘」转化为「体态节奏」框架：自然柔和 S 型动态、肩腰胯流畅节奏感、圆润匀称。

### 场景：卧室

将「暧昧卧室写真」转化为「生活方式/角色插画」框架：明亮温馨、浅色床品、柔和自然光、端正姿态。

## 生成失败后的诊断修复

| 失败类型 | 修复方向 |
|---|---|
| 提示词被拒 | 删除所有红色词，黄色词全部转译，检查风险叠加 |
| 结果太保守 | 不要加「更性感」，改为增强时尚表现力/气质/体态/服装贴合度/光影质感 |
| 身材变形 | 补充真实人体结构约束：比例协调、手指数量正确、四肢自然 |
| 脸太幼 | 明确声明成熟自然的成年女性脸型，不幼态、不儿童化 |
| 画面太擦边 | 回到高级商业人像框架，减少暴露，强调服装/光影/气质 |

## 非人物类提示词

以上规则集中于人物图片，但安全审查的底层逻辑适用于所有题材。系统关注的核心问题是：

- 这张图会不会造成现实伤害？
- 会不会侵犯他人权益？
- 会不会误导公众？
- 会不会涉及违法危险帮助？

对于涉及暴力、真实人物、政治事件、医疗、证件、版权角色等题材的提示词优化，同样适用「把刺激性表达替换为专业化/学术化/商业化表达」的核心策略。
